# Study Buddy App 

How to get started:

```
Open your terminal and then type. 
$ git clone {the url to the GitHub repo} This clones the repo.
cd into the new folder and type. 
$ npm install. This installs the required dependencies.
To run the React project. 
$ npm start.
```
